# AI-Equity-Research-Automation

Automated equity research workflow built using Python, n8n, AI Agents, and Notion.

## Features

- Extract financial data from Screener Excel exports
- Calculate:
  - ROE
  - ROCE
  - Debt/Equity
  - Interest Coverage
  - Price to Book
  - Growth Metrics
- Generate AI-powered research notes
- Store research reports automatically in Notion
- Orchestrate workflow using n8n

## Tech Stack

- Python
- Pandas
- Flask
- n8n
- OpenAI
- Notion API

## Workflow

Excel → Python API → n8n → AI Agent → Notion Database
